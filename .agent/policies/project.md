# project.md

## Repo Purpose

QuantMap identifies the best local inference configuration for a machine/model/workload using measured benchmarking.

## What the System Is

QuantMap is a benchmarking, scoring, and reporting system for local LLM inference.
It runs campaigns, compares configurations, and produces auditable results.

## What Success Means

Success means the repo can:

- run reproducible benchmark campaigns
- compare configs fairly
- preserve trustworthy measurement and reporting
- produce clear results without hiding uncertainty
- help identify the best practical configuration for real use

## Domain

Local LLM performance evaluation on real hardware, emphasizing rigor, reproducibility, and trustworthy reporting.
